实际上使用的是封装后的html的语句

### 1. 标题语法
```	
# 大标题
## 二级标题
```

### 2. 强调语法

```
*斜体*
**加粗**
***加粗的斜体***
```

### 3. 列表语法

```
-  无序列表

1. 有序列表
2. 有序列表
```

### 4. 引用语法

```
>引用
>>引用
>
>块引用
```

### 5. 代码语法

`代码`

	```python
	#代码块
	import math
		a = sqrt(4)
		print(a)
	```

### 6. 分割线语法

`**********（多个）`

### 7. 链接语法

#### 基本链接
```
[超链接显示名](超链接地址 "超链接title")
链接title是当鼠标悬停在链接上时会出现的文字
%20 代替空格
```
#### 网址链接

`<www.bing.com>`


#### 引用格式链接
```
[百度][1] 

[1]: <https://www.baidu.com> "一个百科网站"
```
